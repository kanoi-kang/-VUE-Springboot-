```java
// JwtService.java
public String generateToken(Long userId, String username) {
    Map<String, Object> claims = new HashMap<>();
    claims.put("userId", userId);
    claims.put("username", username);
    
    return Jwts.builder()
            .claims(claims)
            .subject(String.valueOf(userId))
            .issuedAt(new Date())
            .expiration(new Date(System.currentTimeMillis() + 86400000)) // 24小时
            .signWith(key)
            .compact();
}

public boolean validateToken(String token) {
    try {
        Jwts.parser()
                .verifyWith(key)
                .build()
                .parseSignedClaims(token);
        return true;
    } catch (Exception e) {
        return false;
    }
}
```

#### 1.2.3 购物车模块

**购物车状态管理**（前端）：

```typescript
// stores/cart.ts
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'

export const useCartStore = defineStore('cart', () => {
  const items = ref<CartItem[]>([])
  
  const totalPrice = computed(() => {
    return items.value
      .filter(item => item.selected)
      .reduce((sum, item) => sum + item.price * item.quantity, 0)
  })
  
  const totalCount = computed(() => {
    return items.value.reduce((sum, item) => sum + item.quantity, 0)
  })
  
  const addItem = (product: Product, quantity = 1) => {
    const existing = items.value.find(item => item.productId === product.id)
    if (existing) {
      existing.quantity += quantity
    } else {
      items.value.push({
        id: Date.now(),
        productId: product.id,
        name: product.name,
        price: product.price,
        image: product.pic,
        quantity,
        selected: true
      })
    }
    saveToStorage()
  }
  
  const updateQuantity = (id: number, quantity: number) => {
    const item = items.value.find(item => item.id === id)
    if (item) {
      item.quantity = quantity
      saveToStorage()
    }
  }
  
  const removeItem = (id: number) => {
    items.value = items.value.filter(item => item.id !== id)
    saveToStorage()
  }
  
  const saveToStorage = () => {
    localStorage.setItem('cart', JSON.stringify(items.value))
  }
  
  const loadFromStorage = () => {
    const saved = localStorage.getItem('cart')
    if (saved) {
      items.value = JSON.parse(saved)
    }
  }
  
  return { items, totalPrice, totalCount, addItem, updateQuantity, removeItem, loadFromStorage }
})
```

#### 1.2.4 订单模块

**订单创建流程**：

```
1. 验证商品库存和状态
2. 生成订单编号
3. 创建订单记录
4. 创建订单项记录
5. 扣减库存
6. 返回订单信息
```

**关键代码**（后端）：

```java
// OrderService.java
@Transactional
public OrderVO createOrder(Long userId, List<OrderItemRequest> items, 
        Address address, Long couponId) {
    
    // 验证商品
    for (OrderItemRequest item : items) {
        Product product = productRepository.findById(item.getProductId())
            .orElseThrow(() -> new BusinessException(404, "商品不存在"));
        
        if (product.getIsPublish() != 1) {
            throw new BusinessException(400, "商品已下架: " + product.getName());
        }
        if (product.getStock() < item.getQuantity()) {
            throw new BusinessException(400, "库存不足: " + product.getName());
        }
    }
    
    // 计算金额
    BigDecimal totalAmount = items.stream()
        .map(item -> {
            Product p = productRepository.findById(item.getProductId()).get();
            return p.getPrice().multiply(BigDecimal.valueOf(item.getQuantity()));
        })
        .reduce(BigDecimal.ZERO, BigDecimal::add);
    
    // 创建订单
    Order order = new Order();
    order.setOrderNo(generateOrderNo(userId));
    order.setUserId(userId);
    order.setStatus(0); // 待付款
    order.setTotalAmount(totalAmount);
    order.setPayAmount(totalAmount);
    order.setConsignee(address.getConsignee());
    order.setPhone(address.getPhone());
    order.setAddressId(address.getId());
    order = orderRepository.save(order);
    
    // 创建订单项并扣减库存
    for (OrderItemRequest item : items) {
        Product product = productRepository.findById(item.getProductId()).get();
        
        OrderItem orderItem = new OrderItem();
        orderItem.setOrderId(order.getId());
        orderItem.setProductId(product.getId());
        orderItem.setProductName(product.getName());
        orderItem.setProductPic(product.getPic());
        orderItem.setPrice(product.getPrice());
        orderItem.setQuantity(item.getQuantity());
        orderItem.setTotalPrice(product.getPrice().multiply(BigDecimal.valueOf(item.getQuantity())));
        orderItemRepository.save(orderItem);
        
        // 扣减库存
        product.setStock(product.getStock() - item.getQuantity());
        productRepository.save(product);
    }
    
    return convertToVO(order);
}

private String generateOrderNo(Long userId) {
    String timestamp = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMddHHmmss"));
    String random = String.format("%06d", new Random().nextInt(1000000));
    return timestamp + random + (userId % 10000);
}
```

#### 1.2.5 支付模块

**支付流程**：

```
创建订单 → 发起支付 → 调用支付接口 → 返回支付链接/二维码 → 用户支付 → 支付回调 → 更新订单状态
```

**支付回调处理**（后端）：

```java
// PaymentController.java
@PostMapping("/callback/{payType}")
public Result<Void> handleCallback(@PathVariable String payType, 
        @RequestBody PaymentCallbackRequest request) {
    
    // 验证签名
    if (!verifySignature(request)) {
        return Result.error(400, "签名验证失败");
    }
    
    // 查询订单
    Order order = orderRepository.findByOrderNo(request.getOrderNo())
        .orElseThrow(() -> new BusinessException(404, "订单不存在"));
    
    if (order.getPayStatus() == 1) {
        return Result.success(null); // 幂等处理
    }
    
    // 更新订单状态
    if ("SUCCESS".equals(request.getTradeStatus())) {
        order.setPayStatus(1);
        order.setPayType(payType);
        order.setPayTime(LocalDateTime.now());
        order.setStatus(1); // 待发货
        orderRepository.save(order);
        
        // 记录支付日志
        Payment payment = new Payment();
        payment.setOrderId(order.getId());
        payment.setPayType(payType);
        payment.setTradeNo(request.getTradeNo());
        payment.setAmount(order.getPayAmount());
        payment.setStatus(1);
        payment.setPayTime(order.getPayTime());
        paymentRepository.save(payment);
    }
    
    return Result.success(null);
}
```

### 3.2 部署步骤

```bash
# 打包项目
mvn clean package -DskipTests

# 运行项目
java -jar target/xjwl-mall-0.0.1-SNAPSHOT.jar
```

**前端部署**：

```bash
# 安装依赖
npm install

# 构建项目
npm run build

# 部署到Nginx
cp -r dist/* /usr/share/nginx/html/
```

**Nginx配置**：

```nginx
server {
    listen 80;
    server_name example.com;
    
    # 用户端
    location / {
        root /usr/share/nginx/html/user-app;
        try_files $uri $uri/ /index.html;
    }
    
    # 管理端
    location /admin {
        root /usr/share/nginx/html/admin-app;
        try_files $uri $uri/ /admin/index.html;
    }
    
    # API代理
    location /api {
        proxy_pass http://localhost:8080/api;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
