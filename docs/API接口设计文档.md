# 新疆文旅电子商城 - API接口文档

---

## 一、文档说明

### 1.1 接口概述

| 项目 | 说明 |
|-----|-----|
| 接口协议 | RESTful API |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |
| 认证方式 | JWT Bearer Token |
| Base URL | http://localhost:8080/api |

### 1.2 统一响应结构

**响应格式**：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {}
}
```

## 二、用户认证模块

### 2.1 用户注册

| 项目 | 内容 |
|-----|-----|
| **接口概述** | 提供用户注册功能，适用于新用户创建账号场景 |
| **功能说明** | 用户通过用户名、密码、手机号等信息注册成为平台用户 |
| **适用场景** | 用户首次使用系统，需要创建新账号 |
| **调用方式** | POST /user/register |

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 说明 |
|---------|------|---------|-----|
| username | String | 是 | 用户名，4-20位字母数字组合 |
| password | String | 是 | 密码，6-20位 |
| phone | String | 是 | 手机号，11位数字 |
| email | String | 否 | 邮箱地址 |

**请求示例**：
```json
{
  "username": "user123",
  "password": "123456",
  "phone": "13800138000",
  "email": "user@example.com"
}
```

**返回结果**：
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "id": 1,
    "username": "user123",
    "nickname": "用户123"
  }
}
```

**错误码定义**：

| 错误码 | 含义 | 解决方案 |
|-------|-----|---------|
| 400 | 用户名已存在 | 请更换用户名后重试 |
| 400 | 手机号已注册 | 该手机号已绑定其他账号 |
| 400 | 参数格式错误 | 检查请求参数格式是否符合要求 |

---

| 项目 | 内容 |
|-----|-----|
| **接口概述** | 提供用户登录功能，适用于用户进入系统时的身份认证 |
| **功能说明** | 用户通过用户名和密码验证身份，获取JWT Token用于后续接口访问 |

**返回结果**：

| 错误码 | 含义 | 解决方案 |

## 三、商品模块

### 3.1 获取商品列表

| 项目 | 内容 |
|-----|-----|
| **接口概述** | 获取商品列表，支持分页、筛选和搜索 |
| **功能说明** | 获取已上架商品列表，支持按分类、品牌、关键词筛选，支持分页 |
| **适用场景** | 商城首页商品展示、商品列表页、搜索结果页 |
| **调用方式** | GET /goods/products |

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 说明 |
|---------|------|---------|-----|
| page | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页数量，默认10，最大100 |
| categoryId | Long | 否 | 分类ID，筛选指定分类 |
| brandId | Long | 否 | 品牌ID，筛选指定品牌 |
| keyword | String | 否 | 搜索关键词，模糊匹配商品名 |
| isPublish | Integer | 否 | 是否发布，默认1（已发布） |
| sortBy | String | 否 | 排序字段：price、createTime、sales |
| sortDir | String | 否 | 排序方向：asc、desc |

**请求示例**：
```
GET /goods/products?page=1&pageSize=20&categoryId=1&keyword=和田玉枣
```

**返回结果**：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": 1,
        "name": "新疆和田玉枣",
        "productSn": "PD20240001",
        "price": 99.00,
        "originalPrice": 128.00,
        "pic": "https://.../image1.jpg",
        "images": ["image1.jpg", "image2.jpg"],
        "stock": 100,
        "categoryName": "特色干果",
        "brandName": "和田玉枣",
        "isNew": 1,
        "isHot": 1
      }
    ],
    "total": 50,
    "page": 1,
    "pageSize": 20,
    "totalPages": 3
  }
}
```

**错误码定义**：

| 错误码 | 含义 | 解决方案 |
|-------|-----|---------|
| 400 | 参数格式错误 | 检查page、pageSize等参数是否为有效数字 |
| 500 | 服务器内部错误 | 请稍后重试或联系技术支持 |

## 五、订单模块

### 5.1 创建订单

| 项目 | 内容 |
|-----|-----|
| **接口概述** | 将购物车商品提交为正式订单 |
| **功能说明** | 选择收货地址和优惠券，生成正式订单，扣减库存 |
| **适用场景** | 购物车结算页点击"提交订单" |
| **调用方式** | POST /orders |
| **认证要求** | 必须登录 |

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 说明 |
|---------|------|---------|-----|
| addressId | Long | 是 | 收货地址ID |
| couponId | Long | 否 | 优惠券ID，不使用则填null |
| items | Array | 是 | 订单商品列表 |
| items[].productId | Long | 是 | 商品ID |
| items[].quantity | Integer | 是 | 购买数量 |
| remark | String | 否 | 订单备注 |

**请求示例**：
```json
{
  "addressId": 1,
  "couponId": null,
  "items": [
    {
      "productId": 1,
      "quantity": 2
    }
  ],
  "remark": "请尽快发货"
}
```

**返回结果**：
```json
{
  "code": 200,
  "message": "订单创建成功",
  "data": {
    "orderId": 1,
    "orderNo": "202401151030256789123456",
    "payAmount": 198.00,
    "expireTime": "2024-01-15 11:30:25"
  }
}
```

**错误码定义**：

| 错误码 | 含义 | 解决方案 |
|-------|-----|---------|
| 400 | 商品已下架 | 请检查订单中的商品状态 |
| 400 | 库存不足 | 商品库存无法满足订单，请修改数量 |
| 400 | 收货地址不存在 | 请选择有效的收货地址 |
| 400 | 优惠券不可用 | 优惠券不满足使用条件 |
| 404 | 商品不存在 | 商品可能已下架或删除 |
| 401 | 未授权 | 请先登录 |

```json
{
```

## 十、公共错误码

| 错误码 | 含义 | 解决方案 |
|-------|-----|---------|
| 200 | 操作成功 | - |
| 400 | 请求参数错误 | 检查请求参数格式和必填项 |
| 401 | 未授权 | 请登录后重试，Token可能已过期 |
| 403 | 禁止访问 | 无权限执行此操作 |
| 404 | 资源不存在 | 检查请求的资源ID是否正确 |
| 409 | 资源冲突 | 可能已存在相同数据 |
| 429 | 请求过于频繁 | 请稍后再试 |
| 500 | 系统内部错误 | 请稍后重试或联系技术支持 |
