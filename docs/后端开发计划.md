## ✅ 电子商城后端开发计划（最终版）

### 📋 项目技术栈
- **框架**：Spring Boot 3.5.14
- **Java版本**：17
- **数据库**：MySQL 8.0 + Spring Data JPA
- **缓存**：Redis
- **前端**：Vue（商城用户端 + 管理后台独立项目）
- **支付**：微信支付 + 支付宝
- **安全**：JWT + Spring Security

### 📝 开发阶段详细计划

#### 🔧 第一阶段：项目基础架构（1-2天）

| 序号 | 任务 | 说明 |
|------|------|------|
| 1 | 验证JDK 17环境 | 确保编译通过 |
| 2 | 添加Redis依赖配置 | spring-boot-starter-data-redis |
| 3 | 全局异常处理 | @ControllerAdvice |
| 4 | 跨域配置 | CORS配置 |
| 5 | API响应封装 | Result<T>统一返回 |
| 6 | 分页封装 | PageResult<T> |
| 7 | Swagger文档 | springdoc-openapi |

---

#### 👤 第二阶段：用户管理模块（2-3天）

**实体设计**：
- `User` - 用户表
- `Address` - 收货地址表
- `Role` - 角色表

**API接口**：
| 接口 | 方法 | 说明 |
|------|------|------|
| /api/users/register | POST | 用户注册 |
| /api/users/login | POST | 用户登录 |
| /api/users/me | GET | 个人信息 |
| /api/users/password | PUT | 修改密码 |
| /api/addresses | CRUD | 收货地址 |

---

#### 📦 第三阶段：商品模块（3-4天）

**实体设计**：
- `Category` - 商品分类表（树形结构）
- `Brand` - 品牌表
- `Product` - 商品表
- `ProductImage` - 商品图片表
- `ProductSpecification` - 商品规格表

**API接口**：
| 接口 | 说明 |
|------|------|
| /api/products | 商品列表（分页+筛选） |
| /api/products/{id} | 商品详情 |
| /api/products/search | 关键词搜索 |
| /api/categories | 分类树 |
| /api/products/hot | 热门商品 |

**Redis缓存**：
- 商品详情缓存（key: product:{id}）
- 分类列表缓存（key: categories:tree）

---

#### 🛒 第四阶段：购物车与订单模块（3-4天）

|------|------|

### 验证码验证流程

1. **前端调用验证码接口** → 返回PNG图片，验证码存储在Session中
2. **用户填写表单** → 包含username、password、captcha字段
3. **后端验证** → 比对Session中的验证码，验证成功后清除Session
